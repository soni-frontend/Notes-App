# Notes App

A modern and responsive Notes App built using HTML, CSS, and JavaScript.  
This application allows users to create, edit, delete, and manage notes easily while storing data in the browser using Local Storage.

## Features

- Create and save new notes  
- Edit existing notes anytime  
- Delete notes when no longer needed  
- Automatically save notes using Local Storage  
- Clean and user-friendly interface  
- Fully responsive design for mobile and desktop devices  

## Technologies Used

- HTML5  
- CSS3  
- JavaScript  
- Local Storage API  

## Project Purpose

This project was created to improve my frontend development skills and strengthen my understanding of JavaScript concepts such as DOM Manipulation, Event Handling, and Local Storage.
